# Samuel Balogh Portfolio

Minimal cyberpunk portfolio site for Samuel Balogh, focused on AI systems, RAG, AWS, Python, and full-stack consulting work.

## Live Site

GitHub Pages:

```text
https://samuelbalogh.github.io/mome-portfolio/
```

## Local Preview

This is a static site with no build step. Open `index.html` directly in a browser.

```text
index.html
```

## Files

- `index.html` - page structure and portfolio content
- `styles.css` - responsive layout, dark/light themes, cyberpunk visual system
- `script.js` - theme toggle and reveal animations
- `.github/workflows/pages.yml` - GitHub Pages deployment workflow

## Deployment

The site deploys through GitHub Actions when changes are pushed to `main`.

In the repository settings, GitHub Pages should use:

```text
Build and deployment -> Source -> GitHub Actions
```
